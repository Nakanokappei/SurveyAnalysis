## 1. セットアップ

1. Windows 10/11 に [.NET 8 SDK](https://dotnet.microsoft.com/download) をインストール
   します。
2. アプリをビルドして起動するか（README の「ビルドと実行」を参照）、単体実行ファイル
   `SurveyAnalysis.exe` を起動します。
3. サイドバー下部の **設定** を開き、**LLM** タブで OpenAI 互換の **API キー** を貼り付け、
   モデルを選びます。これは OCR・感情・トピック機能に必要です。CSV 取り込みや手入力は
   キーなしでも使えます。

## 2. プロジェクトを作る

ようこそ画面（**プロジェクトを開始しましょう**）から:

- **プロジェクトを作る** — 空のプロジェクトを作り、項目を自分で定義します。
- **CSV からプロジェクトを作る** — CSV を選ぶとヘッダーから項目を推定し、その後で調整
  できます。

最近使ったプロジェクトは一覧の **開く** から再度開けます。

## 3. 項目を設定する

サイドバーのプロジェクト名の横にある **構成** を開きます。各行が 1 つの項目です:

- **項目名** — 列 / 設問のラベル。
- **データ型** — 氏名・性別・住所・電話番号・メールアドレス・日付・選択肢・数値・
  テキスト・テキスト（改行あり） のいずれか。先頭の 5 つは**個人情報**で、保存時に暗号化
  されます（🔒 バッジが付きます）。
- **分析方法** — なし・感情極性分析・トピック割り当て。感情/トピックは自由記述の項目に
  適用します。

選択肢の項目は、1 つのセルに複数の選択をセミコロン `;` 区切りで持てます
（例: `テレビ;インターネット`）。

## 4. 回答を取り込む

### CSV 取り込み

サイドバーの **インポート (CSV)** を使います。行をプレビュー・ページ送りして確認し、
取り込みます。UTF-8 / Shift-JIS の両方に対応しています。

### スキャン用紙の取り込み（OCR）

1. **画像を読み込む** で 1 枚以上のスキャン画像を選ぶか、**フォルダから画像を読み込む** で
   フォルダ内の全画像を読み取ります（API キーが必要 — セットアップ参照）。
2. 各画像は Vision モデルで仮テーブルへ読み取られます。画像ごとに課金リクエストが発生
   するため、最初にまとめて確認を求めます。
3. **校正画面** が開きます: 左にスキャン画像、右に読み取り値（編集可能）。誤読を修正
   します。
   - **選択肢** の項目では、複数選択を半角セミコロン `;` で区切る旨の注記が出ます。
4. レコードごとに **この内容で取り込む** か **破棄** を選びます。取り込みを確定するまで
   実データには書き込まれません。途中まで校正したバッチは安全に保持され、次回続けられ
   ます。
5. 既存の回答と完全に一致するレコードは、追加で取り込むか・無視するか・キャンセルするかを
   尋ねられます。

## 5. ダッシュボード

**ダッシュボード** は選択中の期間の回答を要約します:

- **総回答数**・**ネガティブ件数**・**平均感情スコア**。
- **トピック別 件数** と **感情極性の分布**。
- **回答一覧（抜粋）**。

右上の **対象期間** ピッカーで全体を期間で絞り込めます。

## 6. 切り口（分析）

サイドバーの **切り口** で、シェブロンから軸を展開して内訳を開きます:

- **期間別** — 時系列の感情極性の推移グラフ。**点をクリックするとその日 / 週に期間を
  絞り込みます。**
- **曜日別**・**地域別** — 表形式の内訳。
- **トピック別**・**選択肢別** — 設問ごとにサブ項目が 1 つずつ。

各軸は設問とクロス集計できます（軸 × 設問）。同じ **対象期間** ピッカーで切り口も絞り込め
ます。

## 7. エクスポート

- **CSV エクスポート** — 現在のビュー / 回答を CSV に出力します。
- **エクスポート** — 月次レポートを PDF で生成します。

## 8. 設定

サイドバー下部の **設定** を開きます:

- **LLM** — OCR・感情・トピック機能で使う API キーとモデル。
- **メール / SMTP** — レポート送信用の Gmail / SMTP 認証情報。

ここで入力した秘密情報は保存前に暗号化されます（下記参照）。設定のすぐ下にある
**バージョン情報** では、バージョン・MIT ライセンス・リポジトリへのリンクを確認できます。

## 9. データの保存とプライバシー

- データはすべてローカルの SQLite データベース
  `%LOCALAPPDATA%\SurveyAnalysis\surveyanalysis.db` に保存されます。
- 個人情報の項目（氏名・性別・住所・電話番号・メールアドレス）と保存される秘密情報
  （API キー・SMTP / Gmail パスワード）は Windows DPAPI（現在のユーザースコープ）で保存時
  暗号化されるため、DB ファイル単体では中身を読み取れません。
- 設定した LLM API への明示的なリクエスト（OCR・感情・トピック）以外に、端末から送信される
  ものはありません。
- すべてリセットするには、アプリを閉じて DB ファイルを削除します。

## 10. トラブルシューティング

- **「画像の読み取りには OpenAI の API キーが必要です」** — **設定 → LLM** でキーを設定して
  ください。
- **OCR がチェック欄を誤読する** — 取り込み前に校正画面で修正してください。読み取り値は
  常に確認用に表示されます。
- **別の PC に移したら秘密情報が空になる** — DPAPI 暗号化値は書き込んだ Windows ユーザーに
  紐づくため他環境では復号できません。設定で入力し直してください。
