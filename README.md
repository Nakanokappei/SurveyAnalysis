# SurveyAnalysis (アンケート分析)

A Windows desktop application for analyzing survey responses — import from CSV or
scanned paper forms (OCR), then explore sentiment, topics, and cross-tabs.

アンケート回答を分析する Windows デスクトップアプリ — CSV やスキャン用紙（OCR）から
取り込み、感情・トピック・クロス集計で分析します。
